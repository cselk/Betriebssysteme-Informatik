"Darf ein bestimmter Teilabschnitt eines nebenläufigen Programms nur von einem Thread gleichzeitig durchlaufen werden, wird dies als kritischer Abschnitt bezeichnet." Bei kriteischen Abschnitten würde ein wechselseitiger Ausschluss notwendig. 
Unter Wechselseitigen Ausschluss versteht man Verfahren die verhindern, dass zwei Programme parallel auf eine Ressource zugereifen bzw. einen kritischen Abschnitt durchlaufen und so Komplikationen entstehen würden.
