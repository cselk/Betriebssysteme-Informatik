### Styleguide
Ihr erstellt für jeden eurer Tasks ein MarkDown(.md) file, der den Namen eures Tasks enthält. Zu jedem Thema, wird zusätzlich ein Übersichts Task erstellt. Der Übersichtstask, enthält die Inferationen aus den zu dem Thema gehörigen einzeltasks und wird in dem index.md verlinkt.

### Support Material Sammlung
Es gibt einen Material Sammlungsordner. Wenn ihr Bilder oder Quellen oder ähnliches verwenden wollt, stellt stellt diese in den src-ordner. Ihr verlinkt auf diese Materialen oder bettet diese in euren Markdowns ein.
